# Contributing

volumeHUD is not accepting pull requests or feature requests.

This is a personal project that I built for my own use and share freely. I consider it feature-complete. The code is open source under MIT, so you're welcome to fork it and take it in whatever direction you want, but I'm not merging external changes or expanding scope.

Bug reports are welcome, but please note that I cannot provide support for specific hardware configurations or interactions with third-party apps. If you've found a genuine bug, please check existing issues and make sure you've tried a clean reinstall (see [README](./README)). I'll do my best to look into it when I can.

Thank you for understanding, and thank you for using the app!
